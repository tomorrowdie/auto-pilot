# Database Schema: Product Intelligence Engine (Rufus)

This schema is designed to store the "Insights" generated by the AI Auditors and Analysts, allowing for automated strategy generation and historical tracking of product improvements.

## Table: `product_intelligence`

| Column Name | Data Type | Constraints | Description |
| :--- | :--- | :--- | :--- |
| **sku** | `VARCHAR(50)` | `PRIMARY KEY` | Unique Product Identifier (e.g., "VIKING-MUG-01"). |
| **analysis_id** | `UUID` | `DEFAULT uuid_generate_v4()` | Unique ID for this specific analysis run. |
| **created_at** | `TIMESTAMP` | `DEFAULT NOW()` | When this analysis was performed. |
| **customer_profile** | `JSONB` | `NULLABLE` | **From P1 Analyst:** Target persona, key desires, and fears. <br>Example: `{"persona": "D&D Player", "desire": "Immersion", "fear": "Leaking"}` |
| **product_weaknesses** | `JSONB` | `NULLABLE` | **From P1 & P2 Auditors:** List of logic gaps, contradictions, and missing specs found by "Bad Cops". <br>Example: `["Unsealed seams", "Ambiguous material grade"]` |
| **marketing_hooks** | `JSONB` | `NULLABLE` | **From P2 Analyst:** Validated selling points and "Hero" identity. <br>Example: `{"core_identity": "Conversation Starter", "hooks": ["Giftable", "Heavy-duty"]}` |
| **trap_questions** | `JSONB` | `NULLABLE` | **Aggregated:** All generated trap/validation questions for future regression testing against Rufus. |
| **strategy_verdict** | `TEXT` | `NULLABLE` | **From Master Agent:** The final "Go/No-Go" decision or "Fix-Required" summary. |
| **listing_optimization** | `TEXT` | `NULLABLE` | **From Master Agent:** Specific copy suggestions (Title, Bullets) generated to counter the weaknesses. |
| **creative_strategy** | `JSONB` | `NULLABLE` | **From Master Agent (Creative War Room):** PPC targets and content topics. <br>Example: `{"ppc_targets": ["Dungeon Masters", "Groomsmen"], "content_topics": ["Viking Wedding Guide"]}` |

### Indexes
* `CREATE INDEX idx_pi_sku ON product_intelligence(sku);`
* `CREATE INDEX idx_pi_created_at ON product_intelligence(created_at);`
